About YAETM
***********************
Ein SW-System verwaltet das Ersatzteilelager einer Autowerkstatt.


Features
********
Die Benutzer des SW-Systems können:
• eine bestimmte Anzahl eines Ersatzteils am aktuellen Datum entnehmen
• eine bestimmte Anzahl eines Ersatzteils für eine spätere Entnahme reservieren
• eine Reservierung rückgängig zu machen
• eine bestimmte Anzahl eines Ersatzteils ins Lager einfüllen

Auf dem SW-System können:
• Ersatzteile gesucht werden
• neue Ersatzteile hinzugefügt werden
• Ersatzteile entfernt werden, sofern sie nicht mehr auf Lager sind
